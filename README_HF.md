# Weather Web Application

A modern weather application built with Java Servlets and Jakarta EE.

## Features

- Search weather by city name
- Real-time weather data from OpenWeatherMap API
- Modern glassmorphism UI design
- Responsive layout

## How It Works

Enter a city name in the search bar and get real-time weather information including temperature, humidity, wind speed, pressure, and visibility.
